# UI/UX (PCPartPicker feel)

## Layout
Left rail: Category tree with status chips ✅/❗/—
Center: Eligible parts list (filters + compare)
Right rail: Build summary
  - completion %
  - missing list
  - cheapest vendor mix
  - warnings/errors panel

## Micro-interactions
- When a category flips to ✅, show a tiny celebratory tick (no casino nonsense).
- Sticky save bar with estimated total.
- Compare drawer for parts.

## Accessibility
Keyboard nav through tree + list, visible focus rings, contrast checks.
